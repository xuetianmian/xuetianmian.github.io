1. 对象的类型

对象，其实就是一个无序键值对集合。javascript中有许多的对象子类型，通常称为内置对象。

  • String
  • Number
  • Boolean
  • Object
  • Function
  • Array
  • Date
  • RegExp(正则表达式)
  • Error 当声明一个string,number的基本类型变量时，虽然是基本类型，但是因为JavaScript有对应的内置对象类型，因此在有必要时会自动转换成对应的对象，所以仍可以直接在变量上访问属性或者方法，如：

3. 对象属性描述符和遍历

有时候需要遍历对象的属性，但是不是所有的属性都是可枚举的。
configurable：是否可以可以删除，默认true。
enumerable：是否可枚举，如果可以就能通过for-in循环返回，默认true。
writeable：是否可以修改，默认true。
value，属性值，默认undefined。

若要修改，可以使用Object.defineProperty()方法。

3.1 属性的存在

判断对象是否存在某个属性，有两中方法：in 和 hasownproperty()

两者的区别是，in 会检测属性是否存在对象及其原型链中，如果myObject中没有b属性，但是其原型链中，in检测仍会返回true。hasownproperty()只会检测当前对象。
