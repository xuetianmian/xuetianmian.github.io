发表于 2017-11-18

1. 值的类型

JavaScript 的值有六种主要内置类型，包括：
五种基本类型：undefined，null,boolean,number,string
复合数据类型：object（对象，无序的名值对 集合）

基本类型并不是对象，可以用typeof来查看值的类型
基本类型检测使用 typeof。对象类型检测使用 instanceof。null 或 undefined 的检测使用 == null。

JavaScript中有6个假值，0，NaN,undefined，null,false,’’(空字符串)，null是其中唯一一个typeof会返回“object”的基本类型值。

2. 类型值之间的转换

2.1 Boolean值

数据类型 转换为true的值 转换为false的值
Boolean true false
String 任何非空字符串 空字符串
Number 非0数字（包括无穷大） 0和NaN
Object 任何对象 null
Undefined undefined

总结来说，JavaScript中五中基本数据类型undefined、null、布尔型、数字和字符串，外加对象共六中类型中，只有undefined、null、布尔值中的false、数字中的0和NaN，和字符串中的空字符串’’,共计6种值被转换成false, 其余的都被转换成true。
所以说，JavaScript有以上6个假值，其余为真值。
